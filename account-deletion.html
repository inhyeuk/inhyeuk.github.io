계정 및 데이터 삭제 안내

D.Well 앱에서 계정 및 개인정보를 삭제할 수 있습니다. 계정 삭제 시 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다.

1. 계정 삭제 방법

방법 1: 앱 내에서 삭제 (권장)

1 D.Well 앱 실행
2 설정 메뉴로 이동

홈 화면 → 설정 (⚙️) 아이콘 클릭

3 계정 관리 선택

설정 → 계정 관리

4 계정 삭제 버튼 클릭

"계정 및 데이터 삭제" 버튼 클릭

5 확인 및 삭제

경고 메시지 확인 후 "삭제" 버튼 클릭

방법 2: 이메일로 요청

앱에 접근할 수 없는 경우, 이메일로 계정 삭제를 요청할 수 있습니다.

📧 계정 삭제 요청 이메일

수신: support@yourdomain.com

제목: [계정 삭제 요청] D.Well

내용 포함 사항:

  • 등록된 이메일 주소 (있는 경우)
  • 기기 정보 (Android 모델명)
  • 삭제 사유 (선택사항)

처리 기간: 요청 후 7일 이내 처리

2. 삭제되는 데이터

데이터 유형 삭제 여부 비고
계정 정보 ✅ 즉시 삭제 이메일, 프로필 정보

3. 계정 삭제 전 유의사항

⚠️ 중요: 삭제 전 반드시 확인하세요!

  • 포인트 출금: 미출금 포인트는 복구할 수 없습니다. 삭제 전 반드시 출금하세요.
  • 영구 삭제: 계정 삭제 후 복구가 불가능합니다.
  • 재가입: 동일한 기기로 재가입 시 새로운 계정으로 시작됩니다.
  • 처리 시간: 삭제 요청 후 즉시 처리되며, 법령에 따라 일부 데이터는 보관될 수 있습니다.

4. 삭제 처리 절차

즉시 삭제 (앱 내 삭제)

  • 계정 삭제 버튼 클릭 즉시 처리
  • 로그아웃 및 앱 재시작
  • 개인정보 즉시 삭제

이메일 요청 시

  • 접수: 요청 접수 후 1일 이내 확인 이메일 발송
  • 본인 확인: 필요시 추가 정보 요청
  • 삭제 처리: 본인 확인 후 3일 이내 처리
  • 완료 알림: 삭제 완료 후 이메일 발송

6. 데이터 다운로드

계정 삭제 전 개인 데이터를 다운로드할 수 있습니다:

  • 앱 설정 → 내 데이터 → 데이터 다운로드
  • 다운로드 가능 데이터: 설문 참여 내역, 포인트 적립/출금 내역
  • 형식: JSON 또는 CSV

7. 광고 ID 재설정

계정 삭제 없이 광고 추적만 중단하고 싶은 경우:

Android

  • "광고 맞춤설정 선택 해제" 활성화
